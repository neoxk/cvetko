# Cvetko App

Mobile app for plant discovery, care tracking, and notifications.

## Quick start

This repo is under initial setup. Development scripts and tooling will be added
in the project foundation steps.

## Scripts

- `npm run lint`: Run ESLint with zero warnings allowed.
- `npm run format`: Format the codebase with Prettier.
- `npm run typecheck`: Run the TypeScript compiler in no-emit mode.
- `npm test`: Run Jest tests.
